Webbläsare
En webbläsare är det verktyg du använder när du surfar på internet. Det finns ett flertal olika sådana. Så som Mozilla Firefox, Safari, Google Chorme och Internet Explorer.
Webbserver
Det är en dator som man hämtar information ifrån när man är inne på en hemsida.
HTML
HTML är kod som man skriver för att skapa innehåll på webbsidor mm.
HTTP
HTTP bygger på ett förfrågan/svars förfarande mellan server och klient. HTTP används på webbläsare som har i syfte att hämta html fil eller annan fil från webbserver.
